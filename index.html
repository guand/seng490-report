AngularJS

AngularJS is our frontend portion of the project. This is an MVC framework for web applications to create more structured, shared, and reusable code. A large library of third party support and community development allows for a wide pool of documentation and pre-built components. We explored two options, AngularJS and EmberJS, but AngularJS was chosen due to familiarity overall.

Python's Django

Django is used for our server-side portion of our web application. This is a REST API built to take requests from clients, and runs SQL commands on the database.Django is a high-level Python Web framework that encourages rapid development and clean, pragmatic design. Built by experienced developers, it takes care of much of the hassle of web development, so you can focus on writing your app without needing to reinvent the wheel. It's free and open source. Ridiculously fast.

PostgresSQL Database

PostgreSQL is what we use for our database. PostgreSQL is a powerful, open source object-relational database system. It has more than 15 years of active development and a proven architecture that has earned it a strong reputation for reliability, data integrity, and correctness.

Eventurist.io

Eventurist is a crowd sourced, event discovery, user generated and user moderated web application.

Stop searching across 20 different websites for events. Instead, post, link, comment, and vote on events to bring the most popular events to your attention, and make local events more visible. Be part of a social hub of event activity around your city! This web application will hold all event information for every city around the world.

Built With You In Mind.

Each page is customized to bring the most information on events to you.

Try It Out Now!

Try it out now!

Be part of a large, event driven, locally interested community. It's live at eventurist.io. Vote, comment, post, and link about events nobody else knows about. Find out about all the interesting events in your city!

Local events you won't find on any other discovery app

The biggest issue with finding events is that not every event ends up online. What will you do if the pub down the road never posted it to the Internet? You'll never miss another event now that the locals can post events you never knew about. Discovery applications can only find so much information online. Eventurist aims to get that information constantly using the power of our userbase, and building strong, local event hub for every city.

Discover events tailored to you

Find events easier thanks to our strong filtering algorithms and powerful tagging system. Search for events based on tags or hashtags, check out the most popular, trending tags. Look at events based on day or set of days to plan your week out. Search by keyword to find specifically that event you've been hearing about all week. Search through the events in a different city whether your travelling or moving. There's plenty of events for you to find here, with many options to help find exactly what you're looking for.

Multiple user features to keep you excited about events

Keep track of the interesting events, or track events you've created. Save all your events to review later, or comment on events to discuss it with the community. Ensure no malicious events get created by commenting and downvoting them as you go. Ensure only the best events get popular and work with the community to build a strong set of events for you local area. Check out your saved events later, and wait for future Google Calendar Integration.
