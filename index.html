AngularJS

AngularJS is our frontend portion of the project. This is an MVC framework for web applications to create more structured, shared, and reusable code. A large library of third party support and community development allows for a wide pool of documentation and pre-built components. We explored two options, AngularJS and EmberJS, but AngularJS was chosen due to familiarity overall.

Python's Django

Django is used for our server-side portion of our web application. This is a REST API built to take requests from clients, and runs SQL commands on the database.Django is a high-level Python Web framework that encourages rapid development and clean, pragmatic design. Built by experienced developers, it takes care of much of the hassle of web development, so you can focus on writing your app without needing to reinvent the wheel. It's free and open source. Ridiculously fast.

PostgresSQL Database

PostgreSQL is what we use for our database. PostgreSQL is a powerful, open source object-relational database system. It has more than 15 years of active development and a proven architecture that has earned it a strong reputation for reliability, data integrity, and correctness.
